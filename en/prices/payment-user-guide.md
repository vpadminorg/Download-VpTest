# Payment User Guide
## 分为3种计划
- Free: 无需编码，一键建站，并将任意静态网站瞬间升级成带管理端功能的动态站点
- PlanA: 1$/Mon, 10\$(~~12\$~~)/Annually, More storage and buildtime with more admin features
- PlanB: 2$/Mon, 20\$(~~24\$~~)/Annually, More storage and buildtime with full admin features
